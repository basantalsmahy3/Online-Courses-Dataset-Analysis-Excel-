# Online Courses Dataset Analysis (Excel)

---

## Project Overview
This project analyzes a dataset of online courses to extract insights on popularity, pricing, and distribution.  
The goal is to understand which courses are most popular, pricing trends, and course distribution by level and year.

---

## Tools & Techniques
- **Microsoft Excel**: Pivot Tables, Charts, Conditional Formatting  
- **Data Analysis**: Calculating totals, averages, top courses  
- **Data Visualization**: Charts showing distribution, pricing trends, and course popularity  

---

## Key Achievements
- Calculated number of courses per subject using Pivot Tables  
- Identified top 10 courses by number of subscribers  
- Computed average course price per subject  
- Analyzed course distribution by level and published year trends  
- Delivered visual insights using charts and tables
